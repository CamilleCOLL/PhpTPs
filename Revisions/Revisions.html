1. Syntaxe de Base & Variables (TP 1)

Balises et Affichage

Le code PHP s'exécute côté serveur. L'extension du fichier doit être .php.

<?php
    echo "Bonjour"; // Affiche du texte
    print "Texte";  // Similaire à echo
?>

<!-- Intégration HTML -->
<h1><?php echo $titre; ?></h1>

Variables et Types

Les variables commencent toujours par $. Le typage est dynamique.

$nom = "Jean";      // String
$age = 25;          // Integer
$prix = 19.99;      // Float
$actif = true;      // Boolean

echo gettype($age); // Vérifier le type

Guillemets

Double quotes (") interprètent les variables. Simple quotes (') affichent le texte brut.

echo "J'habite à $ville";  // Affiche la valeur de $ville
echo 'J\'habite à $ville'; // Affiche littéralement $ville

2. Opérateurs (TP 1)

Arithmétique & Concaténation

// Arithmétique
$sum = 10 + 2;
$mod = 10 % 3; // Modulo (reste de la division)
$pow = 2 ** 3; // Puissance (2 exposant 3)

// Concaténation (le point)
$complet = $prenom . " " . $nom;

Comparaison & Logique

// Comparaison
$a == $b   // Égalité de valeur
$a === $b  // Égalité stricte (valeur ET type)
$a != $b   // Différent

// Logique
$a && $b   // ET
$a || $b   // OU
!$a        // NON

3. Structures Conditionnelles (TP 2)

If / Elseif / Else

if ($note >= 16) {
    echo "Excellent";
} elseif ($note >= 10) {
    echo "Passable";
} else {
    echo "Échec";
}

Switch

Utile pour comparer une variable à plusieurs valeurs précises.

switch ($jour) {
    case "lundi":
        echo "Début de semaine";
        break; // Indispensable pour arrêter l'exécution
    default:
        echo "Autre jour";
}

4. Boucles (TP 2)

For

À utiliser quand on connaît le nombre d'itérations.

for ($i = 1; $i <= 5; $i++) {
    echo "Numéro : $i";
}

While

Tant qu'une condition est vraie.

while ($tentative != $secret) {
    $tentative++;
}

Do...While

S'exécute au moins une fois.

do {
    echo "Menu...";
} while ($choix != 3);

5. Tableaux (Arrays) (TP 3)

Tableaux Indexés

Clés numériques automatiques (commence à 0).

$fruits = ["pomme", "banane"]; // Création
$fruits[] = "orange";          // Ajout à la fin
echo $fruits[0];               // Accès (affiche pomme)

Tableaux Associatifs

Clés nommées (chaînes de caractères), similaire aux objets littéraux JS.

$user = [
    "nom" => "Dupont",
    "age" => 25
];

echo $user["nom"]; // Affiche Dupont

Parcours (Foreach)

La boucle incontournable pour les tableaux.

// Parcours simple (valeur)
foreach ($fruits as $fruit) {
    echo $fruit;
}

// Parcours complet (clé et valeur) - Vital pour associatifs
foreach ($user as $cle => $valeur) {
    echo "$cle : $valeur";
}

Tableaux Multidimensionnels

$classe = [
    ["nom" => "Jean", "note" => 15],
    ["nom" => "Marie", "note" => 18]
];
echo $classe[0]["nom"]; // Accès imbriqué

6. Fonctions de Tableaux (TP 3)

Les fonctions natives indispensables.

Information & Recherche

  • count($tab) : Compte le nombre d'éléments.
  • in_array($val, $tab) : Vérifie si la valeur existe (true/false).
  • array_search($val, $tab) : Trouve la clé d'une valeur.
  • isset($tab['key']) : Vérifie si une clé existe.

Modification

  • array_push($tab, $val) : Ajoute à la fin.
  • array_pop($tab) : Retire le dernier élément.
  • array_unshift($tab, $val) : Ajoute au début.
  • array_shift($tab) : Retire le premier élément.
  • array_merge($t1, $t2) : Fusionne deux tableaux.

Tri

  • sort() : Tri croissant (valeur).
  • rsort() : Tri décroissant (valeur).
  • asort() : Tri croissant en gardant les associations clés/valeurs.

7. Fonctions (TP 4)

Déclaration et Appel

function saluer($nom, $titre = "M.") { // Paramètre avec valeur par défaut
    return "Bonjour $titre $nom";     // Retourne un résultat
}

echo saluer("Dupont"); // Appel

Passage par référence

Permet à une fonction de modifier directement la variable originale.

function ajouterUn(&$nb) { // Notez le &
    $nb++;
}
$x = 5;
ajouterUn($x); // $x vaut maintenant 6

Scope (Portée)

Une variable définie hors d'une fonction n'est pas accessible dedans, sauf déclaration spécifique.

$globalVar = 10;

function test() {
    global $globalVar; // Importation de la variable globale
    echo $globalVar;
}

8. Modularité (TP 4)

Superglobales

Variables automatiques accessibles partout.

  • $_GET : Données d'URL (paramètres).
  • $_POST : Données de formulaire envoyé.
  • $_SERVER : Informations serveur.

Inclusion de fichiers

Permet de découper le code en plusieurs fichiers.

include 'header.php';      // Continue même si le fichier manque
require 'config.php';      // Stoppe tout si le fichier manque
require_once 'lib.php';    // Idem, mais ne charge qu'une seule fois
